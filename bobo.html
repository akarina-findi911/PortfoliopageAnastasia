    BOBO


    Bobo ist ein bekannter Anti-Style-Writer in Bern. Wahrscheinlich handelt es sich um eine Gruppe von Writern statt um eine einzelne Person.
    Anti-Style ist eine Gegenbewegung zur Graffiti-Kultur. Während Graffiti oft als rebellische Anti-Bewegung verstanden wird, sind Anhänger des Anti-Style der Meinung, dass Graffiti seine eigenen Regeln verletzt.
    So entstand die Anti-Style-Bewegung: Sie ist eine Gegendbewegung zur Gegenbewegung.
    Kritiker behaupten zudem, dass die Bilder so aussehen, als könnten sie auch von Kindern gemalt worden sein.
